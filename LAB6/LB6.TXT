1. НЕКОТОРЫЕ ОПЕРАТОРЫ МАТЛАБА, НЕОБХОДИМЫЕ ДЛЯ РЕШЕНИЯ ЗАДАЧ ОПТИМИЗАЦИИ.

fmincon -

находит минимум для скалярной функции нескольких переменных с ограничениями начиная с начального приближения. В общем случае, эта задача относится к нелинейной оптимизации с ограничениями или к нелинейному программированию.

•	x = fmincon(fun,x0,A,b) начинает с точки x0 и находит минимум от х для функции представленной как fun при условии выполнения линейных неравенств A*x <= b. x0 может быть скаляром, вектором или матрицей.

•	x = fmincon(fun,x0,A,b,Aeq,beq) минимизирует fun при условии выполнения линейных равенств Aeq*x = beq, 
а так же A*x <= b. Устанавливается A=[] и b=[] в случае отсутствия неравенств.

•	[x,fval] = fmincon(...) возвращает значение целевой функции fun как решение от х.

•	x = fmincon(fun,x0,A,b,Aeq,beq,lb,ub) определяет набор нижних и верхних ограничений на конструируемые переменные х так, что решение всегда находится в диапазоне lb <= x <= ub. Устанавливается Aeq=[] and beq=[] 
в случае отсутствия равенств.

•	x = fmincon(fun,x0,A,b,Aeq,beq,lb,ub,nonlcon) подчиняет минимизацию определенных в nonlcon fmincon нелинейных неравенств c(x) или равенств ceq(x) такому оптимуму, что c(x) <= 0 и ceq(x) = 0. Устанавливается lb=[] и/илиr ub=[] в случае отсутствия ограничений.

•	x = fmincon(fun,x0,A,b,Aeq,beq,lb,ub,nonlcon,options) проводит минимизацию с оптимизационными параметрами, определенными в структурной опции.


Полная информация о функции fmincon содержится в файле fmincon.doc (папка lab4)


2. ПРИМЕРЫ РЕШЕНИЯ ОПТИМИЗАЦИОННЫХ ЗАДАЧ

	Пусть  задана критериальная функция J двух
переменных u1 и u2
             
              J = .5(u1^2 + u2^2).

и, кроме того, имеется линейтое ограничение на переменные

          u2 = -u1 + 1 .
 
Требуется определить значения скалярных переменных u1 и u2, 
при которых критерий принимает стационарное значение, а также
проиллюстрировать полученное решение графически.
	Линиями уровня критериальной функции J являются 
окружности, причем J возрастает с увеличением размера окружности.
Уравнение  u2 = -u1 + 1 задает прямую линию. Очевидно, что мини-
мальное значение J при удовлетворении указанного ограничения
достигается на окружности, которая касается указанной прямой.
	Получим теперь решение в аналитической форме. 
Гамильтониан H имеет вид:

        H = .5*(u1^2 + u2^2) + v*(u2 + u1 -1).

	Из необходимых условий dH/du1=0, dH/du2=0, dH/dv определяем,
что функции H имеет стационарную точку 

              [u10, u20, v0] = [0.5, 0.5, -0.5].

Этому решению соответствует минимальное значение J: J0=0.25.
	Для графической иллюстрации определим Script-file:

clc;
xord=-2.2:.1:2.2;%интервал изменения u1;
uord=-2.2:.1:2.2;%интервал изменения u2;
[u1,u2] = meshgrid(xord,uord);

J = .5*(u1.^2 + u2.^2);
surfc(u1,u2,J);
disp('Оптимальные значения переменных и критерия')
xord0 = 0.5 
uord0 = 0.5
v=-0.5;
J0=0.25
figure(2)
cm=contour(xord,uord,J,[.0001,J0]); 
clabel(cm);
xlabel('u1')
ylabel('u2')
hold on
plot(xord0,uord0,'*')
text(xord0,uord0,'Solution')
hold on
%Линейное ограничение
uord=-xord + 1;
%График линейной функции
plot(xord,uord);

Воспользуемся теперь для нахождения экстремальной точки средствами
Матлаба. 
Script file имеет вид:

%Выбор начального приближения.   

%Выбор начального приближения.

%Следующая опция указывает, что ограничения-неравенства в нашей задаче отсутствуют и имеется 1 линейное ограничение-равенство.Кроме того, возвращается значение целевой функции (fval)

Задание 1.

          J = .5*(a^(-2)*u1.^2 + b^(-2)*u2.^2)

               g = u2 + m^(-1)*u1 - m^(-1)*c = 0,

средств Матлаба. Вывести график с 'оптимальной линией равного
уровня' и графиком ограничения ее касающимся.
