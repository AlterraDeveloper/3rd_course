Существуют различные представления одной и той же модели линейного
программирования. Мы будем рассматривать каноническую форму вида:

         Минимизировать по x критерий 
 
                  J = f'* x

         при наличии ограничений

                  A*x <= b,  
                    x >= 0,

         где x:nx1; f:nx1; b:mx1; A:mxn 
         вектора и матрица соответствующих 
         размерностей.

Замечание: При использовании средств Матлаба допускается смешанное
представление ограничений - равенств и неравенств. (см ниже 
описание оператора LP).

1. ОПЕРАТОРЫ МАТЛАБА, НЕОБХОДИМЫЕ ДЛЯ РЕШЕНИЯ 
   ЗАДАЧ ЛИНЕЙНОГО ПРОГРАММИРОВАНИЯ.

 linprog   	Linear programming.                   
 	X=linprog(f,A,b) решает здачу линейного программирования:
         
             min f'x    при ограничениях:   Ax <= b 
              x
 	
 	X=linprog(f,A,b,VLB,VUB) определяет множество нижних и верхних 
        границ переменных так, что решения всегда находятся в интервале
        X: VLB < X < VUB.
 
 	X=linprog(f,A,b,VLB,VUB,X0) устанавливает начальное значение для X0.
 
 	X=linprog(f,A,b,VLB,VUB,X0,N) указывает, что первые N ограничений,
        определяемых A и b, являются ограничениями равенствами.
 
 	linprog выводит предупреждающее сообщение если решение неограниченное
        или недопустимое. Предупреждающее сообщение может быть выключено
        с использованием следующей записи:
        X=linprog(f,A,b,VLB,VUB,X0,N,-1).

2. ПРИМЕР РЕШЕНИЯ ЗАДАЧИ ЛИНЕЙНОГО ПРОГРАММИРОВАНИЯ.

Рассмотрим следующую задачу:
        При наличии ограничений 

  1) 4*u1 + 3*u2  <= 12,
  2) 2*u1 + 5*u2  <= 10,
  4) 1*u1 + 0*u2  >= 0,
  5) 0*u1 + 1*u2  >= 0

        максимизировать критерий

           12*u1 + 15*u2.

Решение задачи демонстрирует следующий Script-file:

clear all
clc;
%Задание векторов и матрицы условий для численного
%решения с помощью оператора LP.
f=[-12 -15]';
A=[4 3;2 5;-1 0;0 -1];
b=[12;10;0;0];
%Решение проблемы
disp('Решение проблемы будет иметь вид:')
u0=linprog(f,A,b)
disp('Оптимальное значение критерия:')
J0=f'*u0
u01=[1 0]*u0; u02=[0 1]*u0;

%Задание контурной линии, соответствующей
%оптимальному критерию
xord=0:1:4;%интервал изменения u1;
uord=0:1:3;%интервал изменения u2;

[u1,u2] = meshgrid(xord,uord);
J =-12*u1 - 15*u2;
cm=contour(xord,uord,J,[.0000 J0],'w:'); 
clabel(cm);

xlabel('u1'),ylabel('u2');
title('Solution of problem')

hold on;
%Графическое изображение ограничений
%Линейное ограничение 1
a1=3;b1=4;c1=-12;
uord=-(b1/a1)*xord - c1/a1;
plot(xord,uord,'y');
%Линейное ограничение 2
a2=5;b2=2;c2=-10;
uord=-(b2/a2)*xord - c2/a2;
plot(xord,uord,'b');
%Линейное ограничение 3
plot([0 3],[0 0],'g');
%Линейное ограничение 4
plot([0 0],[0 2],'g');

%Определение точки пересечения кривых 1 и 2;
B=[det([c1 a1;c2 a2])/det([a1 b1;a2 b2]) det([b1 c1;b2 c2])/det([a1 b1;a2 b2])]';
b10=[1 0]*B;   b20=[0 1]*B;

%Изображение точек пересечения линейных ограничений (экстремальных точек).
plot(b10,b20,'oy')
text(b10+.1,b20+.1,'B')
plot(0,2,'oy')
text(0+.09,2+.09,'A')
plot(3,0,'oy')
text(3,0+.09,'C')
%Изображение точки оптимума
plot(u01,u02,'*r')

Замечания:
Ограничения (1) и (2) изображены графически (прямые линии) как
уравнения, полученные заменой неравенств на равенства.
Область значений u1 и u2, удовлетворяющая ограничениям (множество
решений) содержится в многоугольнике OABC. 
Вершины многоугольника O A B C - экстремальные точки.
Пунктирная белая линия соответствует оптимальному значению критерия
                     (J = -42.9).
Оптимальное решение определяется экстремальной точкой
                   B(2.1429, 1.1429).



3. ЗАДАНИЯ К ЛАБОРАТОРНОЙ РАБОТЕ

Для задач линейного программирования, определенных в индивидуальном
задании, требуется:
1) изобразить каждую модель задачи графически в пространстве решений;
2) определить графически экстремальные точки и отметить их на графике
   значками 'o';
3) найти оптимальное решение  численно с помощью средств Матлаба 
   и графически. Отметить найденное решение (u10,u20) на графике 
   значком '*'. 
4) вычислить оптимальное значение критерия и отметить его на графике
   соответствующей контурной линией.
